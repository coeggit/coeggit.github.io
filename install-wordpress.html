Wordpress | Download && Config Wordpress


Untuk yang belum config vm web server dan database bisa lihat Disini


Download worpress
  1. Buka direktori /temp dan download wordpress
    2. cd /tmp && wget https://wordpress.org/latest.tar.gz
  2. Extract wordpress yang telah di unduh
    3. tar -xvf latest.tar.gz
  3. Copy paste wordpress ke /var/www/html/
    4. cp -R wordpress /var/www/html/
  4. Ubah kepemilikan direktori wordpress
    5. chown -R www-data:www-data /var/www/html/wordpress/
  5. Ubah izin file dari direktori wordpress
    6. chmod -R 755 /var/www/html/wordpress/
  6. Buat direktori upload
    7. mkdir /var/www/html/wordpress/wp-content/uploads
  7. Ubah izin direktori upload
    8. chown -R www-data:www-data /var/www/html/wordpress/wp-content/uploads/
  8. Buka browser dan coba jalankan wordpress menggunakan IP web-server
    9. https://server-ip/wordpress
Membuat user dan database di mysql

Masuk ke vm database dan lakukan perintah berikut :

  1. Membuat user di mysql
    2. CREATE USER '{username}'@'{ip web server}' IDENTIFIED BY '{password}';
  2. Membuat database
    3. create database {nama_database};
  3. Memberikan hak akses ke pada user
    4. GRANT ALL PRIVILEGES ON {nama database} .* TO '{username}'@'{ip web server}';
  4. Flush Privileges

    5. Flush Privileges berfungsi untuk memerintahkan flush kepada sistem untuk membaca ulang tanpa harus merestart server mysql.
    FLUSH PRIVILEGES;

Install Wordpress
  1. Akses wordpress di web browser
    2. ip-address/wordpress

    Kemudian pilih Bahasa, saya menggunakan Bahasa Indonesia, lalu lanjutkan

  2. Mulai setup wordpress, pilih ayo
  3. Isi form untuk terhubung ke database. Isi seperti berikut
    • Nama Basis Data : Nama database yang telah di buat
    • Nama Pengguna : Sama user mysql yang telah di buat
    • Sandi : Password mysql yang telah di buat
    • Host Basis Data : IP address vm database yang telah dibuat
    • Prefik Tabel : Biarkan default
  4. Mulai installasi, pilih Jalankan pemasangan
  5. Isi informasi
    • Judul Situs : Isi sesuai kebutuhan
    • Nama Pengguna : Isi sesuai keinginan kita
    • Password : Isi sesuai keinginan kita
    • Email : Isi sesuai email yang kita miliki

    CATATAN : Pastikan kita mengingat Nama Pengguna dan Password. Karena itu yang akan kita gunakan untuk login ke wordpress nantinya
    Pilih Install wordpress

  6. Jika berhasil, maka tampilan akan seperti ini

    7. Lalu pilih Log Masuk

  7. Login wordpress menggunakan username dan password yang telah dibuat
  8. Kita akan disambut dengan tampilan dashboard dari wordpress
  9. Jika kita ingin mengunjungi web wordpress yang kita buat kita bisa pergi ke browser dan mengetik
    10. ip-address/wordpress
  10. Selesai
