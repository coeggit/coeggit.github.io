Download && Config Wordpress

  1. Jalankan Apache dan MySQL di XAMPP. Bagi yang belum mempunyai XAMPP bisa download diatas



  2. Buat database untuk wordpress. Masuk ke web browser lalu ketikan :

    3. localhost/phpmyadmin/

    Pilih Baru, lalu di bagian Basis Data, buat nama database kita, disini saya membuat wordpress1



  3. Download wordpress dari website resminya. Bisa download di link diatas pada bagian Bahan



  4. Kemudian Extract Wordpress di

    5. C:\xampp\htdocs

    Agar mudah ganti namanya sesuai database yang telah di buat, disini saya buat wordpress1



  5. Akses wordpress di web browser

    6. localhost/{nama_folder}/

    Pilih Ayo!



  6. Isi form, jika tidak membuat user pada MySQL di maka bisa di isi seperti di gambar. Lalu pilih Kirim



  7. nstall wordpress, pilih Jalankan pemasangan



  8. informasi, lalu pilih Install Wordpress

    • Judul Situs : Isi sesuai kebutuhan
    • Nama Pengguna : Isi sesuai keinginan kita
    • Password : Isi sesuai keinginan kita
    • Tegaskan (ulangi) Kata Sandi : Jika ada tulisan seperti ini maka kita centang
    • Email : Isi sesuai email yang kita miliki



    Catatan :
    Pastikan mengingat Nama Pengguna dan Password

  9. Jika berhasil menginstall tampilannya akan seperti gambar ini, lali pilih Log Masuk



  10. Login menggunakan username dan password yang telah dibuat



  11. Kita akan disambut dengan tampilan dashboard dari wordpress



  12. Jika kita ingin mengunjungi web wordpress yang kita buat kita bisa pergi ke browser dan mengetik

    13. localhost/nama_folder_wordpress_di_htdocs


  13. Selesai
