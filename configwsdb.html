Config Web Server
  1. Sebelum mulai, kita masuk ke root dan update system terlebih dahulu
    2. apt update


  2. Install apache2
    3. apt install apache2


  3. Cek status apache2
    4. sysemctl status apache2


  4. Setting IP static
    5. nano /etc/netplan/00-installer-config.yaml

    Buat seperti baris berikut :

    • ens33 dhcp nya kita buat true karena kita akan menggunakan dhcp sebagai sumber internet.
    • ens34 ini adalah host only, jadi kita setting untuk dhcp false dan kita masukkan IP address sesuai dengan IP address dari host only vm kita.


  5. Apply netplan
    6. netplan apply
  6. Cek hostname
    7. hostname -I


    Disini terdapat 2 IP address yaitu IP dhcp dan IP host only yang bisa kita gunakan nantinya untuk mengakses wordpress

  7. Cek apache2 di web browser menggunakan IP address yang telah kita cek sebelumnya
    8. https://ip-address


  8. Install php
    9. apt install php php-mysql

    Tambahkan baris berikut :



  9. Cek php menggunakan IP address kita sebelumnya
    10. http://ip-address/info.php


  10. Install mysql client
    11. apt install mysql-client

Config Database
  1. Sebelum mulai, kita masuk ke root dan update system terlebih dahulu
    2. apt update


  2. Install mysql-server
    3. apt install mysql-server


  3. Cek status mysql
    4. systemctl status mysql


  4. Install php
    5. apt install php php-mysql


  5. Konfigurasi php
    6. nano /var/www/html/info.php

    Tambahkan baris berikut :



  6. Setting IP static
    7. nano /etc/netplan/00-installer-config.yaml

    Buat seperti baris berikut :

    • ens33 dhcp nya kita buat true karena kita akan menggunakan dhcp sebagai sumber internet.
    • ens34 ini adalah host only, jadi kita setting untuk dhcp false dan kita masukkan IP address sesuai dengan IP address dari host only vm kita.


  7. Apply netplan
    8. netplan apply
  8. Cek hostname
    9. hostname -I


  9. Cek php menggunakan IP address kita
    10. http://ip-address/info.php


  10. Config mysql
    11. sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf

    Ubah bind-address menjadi IP address vm database, lalu ctrl x dan y lalu enter



  11. Untuk cara install wordpress bisa lihat Disini
